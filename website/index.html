AIDE 是一款开源 AI 原生轻量 IDE：在浏览器中运行 Node.js 环境，结合工作区上下文，让 AI 对你的项目结构、依赖与文件关系有全局理解。

打开 AIDE Star / Fork 部署到你的域名
AI Workspace Context
1// AIDE: project-aware assistant
2import { workspaceContext } from './context'
3const prompt = workspaceContext.toSystemPrompt()
4await ai.send({ system: prompt, user: 'Refactor this' })
5// Result: consistent changes across files
拖拽上传项目文件夹后，AI 会得到： 文件结构、 关键内容 和 依赖关系。

核心特性

工作区上下文

让 AI 理解整个项目，而不是只看当前文件。适合重构、修复、生成跨文件改动。

浏览器内运行 Node.js

基于 WebContainer，在浏览器里执行常见 Node 命令，快速验证生成结果。

多模型接入

支持多种 AI Provider（含本地模型方向），按成本/效果自由切换。

为什么是 AIDE

更少误改

AI 在全局上下文中做决策，减少“只改当前文件导致的连锁 bug”。

更快落地

生成、运行、验证在一个界面完成，原型迭代更快。

开源可控

代码透明、可自部署；便于团队定制提示词、模型与流程。
