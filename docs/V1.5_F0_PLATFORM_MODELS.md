## 验收（F0 GA）

| # | 项 |
|---|-----|
| 1 | 登录用户 **无 API Key** 可切换 ≥6 厂商并完成 Chat |
| 2 | 设置页 **无** BYOK 输入（Ollama 本地 URL 保留） |
| 3 | Free 用户仅可选 economy 模型；超限 429 + 升级引导 |

| 5 | Tab / Agent 平台路径计入同一配额 |
| 6 | 平台 Key 不出现在客户端网络请求（仅 `/api/ai/*`） |
| 7 | `test:local` + `e2e/platform-models.spec.ts` 绿 |
