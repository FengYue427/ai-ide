## 3. F1 — Attach（P0）

| 项 | 说明 |
|----|------|
| 启动 | `handleRunCode` 或专用 **Debug Run** 带 inspect 参数 |
| 传输 | WebContainer 暴露 debug 端口 → 浏览器侧 WS 客户端 |
| UI | `DebugPanel`：未连接 / 已连接 / 已暂停 / 已结束 |
| 状态 | `ideStore`：`debugSession: idle \| running \| paused \| ended` |
| 协作 | Viewer：**禁止** 启动调试、不可设断点 |

**验收**：`console.log` 脚本 + 断点行，能进入 **paused** 且面板显示 reason。

---

## 4. F2 — 断点（P0）

| 项 | 说明 |
|----|------|
| Gutter | Monaco `glyphMargin` · 红点/灰点（未绑定） |
| 模型 | `breakpoints: { path, line, enabled }[]` · localStorage 可选 |
| 同步 | 连接后向 runtime **registerBreakpoints** |
| 多文件 | 与 `files[]` / 工作区 path 一致 |

**验收**：刷新页面后断点仍在；禁用断点变灰且不停留。

---

## 5. F3 — 栈与变量（P0）

| 项 | 说明 |
|----|------|
| Call stack | 帧列表 · 点击跳转编辑器行 |
| Locals | 顶层变量 · 对象一层 `JSON` 预览 |
| Scopes | `local` 优先；`global` P1 |
| 只读 | 不提供修改变量（MVP） |

**验收**：递归函数停在内层帧，栈深度 ≥ 3 可导航。

---

## 6. F4 — 执行控制（P0/P1）

| 命令 | 快捷键（建议） | 优先级 |
|------|----------------|:------:|
| Continue | F5 | P0 |
| Step Over | F10 | P0 |
| Step Into | F11 | P1 |
| Step Out | Shift+F11 | P1 |
| Stop | Shift+F5 | P0 |

与 **Run**、**终端 shell** 互斥：调试中禁止新开 jsh 跑同一进程（或明确 kill）。

## 9. 风险与缓解

| 风险 | 缓解 |
|------|------|
| WebContainer 无 inspect | F1 仅 spike；失败则 MVP 降级为 **日志断点**（`debugger` 语句注入）并文档化 |
| DAP 工期过长 | 首版 CDP 子集 + 适配层接口，便于换 DAP |
| 与 Run 状态冲突 | 单一 `runtimeController` 抽象 |
| 性能 | 变量大树截断 · 与 git diff 截断模式一致 |
