## 4. F1 第一周任务（按路线）

开工后 **只做一个 P0**；F1 目标：骨架 + env + 1 条可演示路径。

### 路线 A · 协作 F1（3～5 天）

1. 信令选型记录（Livekit vs 自建 WS）→ ADR 一段写入 `ROADMAP_V1.1.3_COLLAB.md`
2. Prisma 草案 → [V1.1.3_SCHEMA_DRAFT.md](./V1.1.3_SCHEMA_DRAFT.md) 落地 migration
3. `POST /api/collab/rooms`、`POST /api/collab/rooms/:id/join` 骨架 + 鉴权
4. `.env.example` + [V1.1.3_ENV.md](./V1.1.3_ENV.md)
5. Feature flag `VITE_COLLAB_M1_SIGNAL` 门控新路径

### 路线 B · 网关 F1（3～5 天）

1. 新建 `lib/api/aiGateway/`（`resolveRoute`、`forwardChat`）
2. `POST /api/ai/chat` handler + `dispatch.ts` 注册
3. 请求体：`{ messages, provider?, model?, usePlatformKey: true }`
4. 复用 `consumeAiUsage`；429 与 `usage/ai` 对齐
5. `.env.example` 增加 `PLATFORM_DEEPSEEK_API_KEY` 等
6. Feature flag `VITE_AI_GATEWAY` + 设置页占位开关（可先 hidden）
