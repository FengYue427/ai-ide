## 3. 第一本地阶段 F1（非对外发版）✅

**F1 出口**：API 返回 `queued`；**仍不 push 生产叙事为 v1.1.2**。

## 4. 第二本地阶段 F2（执行引擎）✅

1. ✅ `GET|POST /api/jobs/process` + Cron（`*/5 * * * *`）+ `isCronAuthorized`  
2. ✅ `backgroundJobProcessor` + `backgroundJobRunner`（`dummy` 默认；`agent` 占位失败）  
3. ✅ `progress` JSON、`failStaleRunningBackgroundJobs`（30min 超时）  
4. ✅ `npm run jobs:process` · `npm run jobs:verify-cron`  

**F2 出口**：dev 下 dummy 任务经 Cron/脚本可 `succeeded`。

## 5. 第三本地阶段 F3（客户端闭环）✅

1. ✅ `VITE_BACKGROUND_AGENT=true` 时显示 `BackgroundJobsPanel`（右栏 Tab + 工具栏入口）  
2. ✅ 5s 轮询 + 取消  
3. ✅ Chat Agent「后台运行」提交当前上下文  
4. ✅ i18n / 样式  

**F3 出口**：关 Tab 再打开，面板可见终态（dev + Cron）。

## 6. 第四本地阶段 F4（商业与回写）✅

1. ✅ Free：每日 2 次 / 同时 1 个；Pro+：100/日、5 并发（`429` + 升级文案）  
2. ✅ Worker 成功 → 云工作区 `upsert`（`pendingChanges` + `cloudWriteback`）

**F4 出口**：黄金路径 §4 可演示（Pro 用户 + `npm run jobs:process`）。
