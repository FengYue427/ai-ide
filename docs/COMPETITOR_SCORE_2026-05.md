## 相对 Cursor：三条「可赢」与三条「不追」

### 可赢（营销 + 产品投入）

1. **浏览器 + 国内订阅 + BYOK**（获客与转化）  
2. **本机文件夹 + 工具 Agent**（已验收，对标 Composer 入门体验）  
3. **轻量仓 + 中文 UI + 模板市场**（非 Cursor 重心）

### 不追（诚实边界，避免资源分散）

1. VS Code 插件兼容  
2. 全语言 DAP 调试器  
3. 与 Cursor 同级的云后台 Agent / 全仓云索引（短期）

### 有选择地追（GA 后 6～12 月）

1. **Electron + 本机终端**（缩小终端/大仓差距）  
2. **Tab / 索引第二档**（2.35 → 2.6，不追 4.0）  
3. **TS/JS LSP 深化**（单语言做深）
