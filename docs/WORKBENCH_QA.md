# Workbench Shell 手工 QA（F1）

> 分辨率建议：**1280×800**。通过后再发 v1.2.2 F1 合并。

| # | 操作 | 期望 |
|---|------|------|
| 1 | 打开 Chat → 再点 Activity Bar「搜索」 | Chat 关闭，搜索列出现在侧栏与编辑区之间 |
| 2 | 搜索已开 → 再点「搜索」 | 搜索列关闭 |
| 3 | 打开预览 → 再打开搜索 | 预览关闭，仅搜索列 |
| 4 | 打开预览 + 右侧 Git（先开 Git 再开预览） | Git 关闭，仅预览列；编辑区仍可见 |
| 5 | 编辑器内「代码审查」 | 审查面板在 auxiliary 列，不与 Chat 同屏 |
| 6 | 命令面板 Ctrl+Shift+P | 全屏遮罩在最上，Esc 关闭 |
| 7 | 打开设置 | Modal 居中，不挡命令面板关闭后的编辑区 |
| 8 | 1280 宽：Auxiliary + 文件树 + 编辑区 | 编辑区宽度 ≥ 约 480px（目测可输入） |
| 9 | 打开设置 → `Ctrl+Shift+P` 命令面板 | 命令面板在设置 Modal **之上** |
| 10 | 触发 Confirm 对话框 | 在 Toast / Modal 之上 |

记录问题到 GitHub Issue，标签 `workbench-shell`。
