## 模型草案

```prisma
model CollaborationRoom {
  id        String   @id @default(cuid())
  code      String   @unique // 6～8 位邀请码
  hostId    String
  name      String?
  status    String   @default("open") // open | closed
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  host    User              @relation(fields: [hostId], references: [id], onDelete: Cascade)
  members CollaborationMember[]

  @@index([hostId])
}

model CollaborationMember {
  id        String   @id @default(cuid())
  roomId    String
  userId    String
  role      String   @default("editor") // host | editor | viewer
  joinedAt  DateTime @default(now())
  leftAt    DateTime?

  room CollaborationRoom @relation(fields: [roomId], references: [id], onDelete: Cascade)
  user User              @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@unique([roomId, userId])
  @@index([userId])
}
```

`User` 模型需增加：

```prisma
collaborationRoomsHosted CollaborationRoom[] @relation(...)
collaborationMemberships CollaborationMember[]
```

## API 与表对应（F1）

| API | 表操作 |
|-----|--------|
| `POST /api/collab/rooms` | 创建 `CollaborationRoom`，`hostId` = 当前用户 |
| `POST /api/collab/rooms/:id/join` | upsert `CollaborationMember` |
| `GET /api/collab/rooms/:id` | 房间元数据 + 成员列表 |
| Token 签发 | 不持久化；Livekit JWT 或短期 join secret |
