# v1.2.3 F3 — Tab 补全 / FIM / 平台 AI

## 目标

- FIM 优先（DeepSeek / Qwen / Ollama）
- 平台 AI（`keyMode=platform` + 已登录）可走 Tab 补全
- 会话级缓存命中 / 延迟 / 路径统计
- 设置：最大行数、防抖、当前路径说明

## 实现要点

| 模块 | 说明 |
|------|------|
| `tabCompletionStrategy.ts` | `fim → platform → chat` 解析顺序 |
| `inlineCompletionService.ts` | 缓存 → FIM → 平台 → 对话回退；`loggedIn` 入参 |
| `platformAiService.fetchPlatformTabCompletion` | 平台密钥 Tab 补全 |
| `inlineCompletionMetrics.ts` | hits/misses/fim/platform/chat/avg latency |
| `inlineCompletionPrefs.ts` | `debounceMs` 120–800 |
| `EditorLayout` / `registerInlineCompletion` | `isAiConfigured(config, loggedIn)` |
| `SettingsCenter` | 防抖、路径、统计重置 |

npm run test:local
