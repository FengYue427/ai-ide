2. `CHANGELOG.md` 新增 `[1.0.3.1]`
3. 本地执行：

```powershell

npm run go-live:preflight
```

### Step B — Vercel 配置核对

按 [V1.0.3_VERCEL_ENV.md](./V1.0.3_VERCEL_ENV.md) 勾选：

- `APP_URL`
- `AUTH_SECRET`
- `DATABASE_URL`
- `CRON_SECRET`（或 `BILLING_CRON_SECRET`）


- `/api/health` 中 `version` 与包版本一致

### Step D — 发包与触发
