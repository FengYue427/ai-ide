# F3 — Chat / Toolbar / Settings 视觉密度优化

> v1.2.2 第三阶段 · 对应 KICKOFF F3

## 目标

在保持可读性的前提下，减少 10–20% 的垂直与水平间距，让更多内容在有限可视区域内呈现。

## 变更详情

### Toolbar (`toolbar.css`)

| 项 | 原值 | 新值 |
|---|---|---|
| `min-height` | 48px | 40px |
| brand logo | 32px | 28px |
| 按钮 padding | `6px 10px` | `4px 8px` |
| 导航文字 | 13px → 12px |
| 版本号 | 11px → 10px |

### Chat (`chat.css`)

| 项 | 原值 | 新值 |
|---|---|---|
| header padding | `14px` | `10px 12px` |
| v2 header padding | `16px 16px 14px` | `12px 12px 10px` |
| message gap | 12px → 8px (v1), 16px → 10px (v2) |
| avatar size | 28px → 24px (v1), 34px → 30px (v2) |
| bubble padding | `10px 12px` → `8px 10px` (v1), `12px 14px` → `10px 12px` (v2) |
| send button | 44px → 38px |
| input padding | `12px 14px` → `10px 12px` |
| v2 input area | box-shadow 缩减, margin `0 12px 12px` → `0 10px 10px` |
| control strip gap | 6px → 4px |
| mode btn icon | 30px → 26px |
| chat chip | 5px 8px → 3px 6px, 11px → 10px |

### Settings (`settings.css`)

| 项 | 原值 | 新值 |
|---|---|---|
| 总宽度 | 1040px → 960px |
| 侧栏宽度 | 280px → 240px |
| header padding | `22px var(--space-6)` → `16px var(--space-5)` |
| body padding | `var(--space-6)` → `var(--space-4)` |
| card padding | 18px → 14px |
| card gap | 18px → 12px |
| 按钮/font-size | 缩减约 1px 各处 |
| toggle 尺寸 | 52×30 → 44×26 |

### Tokens (`tokens.css`)

- `--toolbar-height`: 48px → 40px
- `--right-panel-width-chat`: 380px → 360px
- `--right-panel-width-chat-wide`: 420px → 400px

## 效果

- **Toolbar** 高度减少约 17%
- **Chat** 消息间距减少约 30%
- **Settings** 总体积减少约 15–20%
- 窄屏下保留可读性（font-size 未低于 9px）
