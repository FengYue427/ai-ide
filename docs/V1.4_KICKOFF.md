## 1. 北极星

v1.3 把「仓库理解 + 多语言导航 + Agent 可靠」抬到 **~3.3**；v1.4 用七个阶段补齐 **v1.3.x 刻意搁置** 的基础能力，把综合分推到 **≥3.35 → 3.4**，再评估宣传线。

1. **Tab/FIM 生产级**：延迟可测、生产默认策略可运营  
2. **大仓索引**：2k Worker / 分片重建入门  
3. **Git 日常面板**：status 矩阵 + hunk stage 完整流  
4. **桌面壳深化**：Electron 默认路径与开箱体验  
5. **后台 Agent**：生产默认可用（非仅设置卡）  
6. **MCP/插件生产化**：官方目录 + KB 深化  
7. **GA**：v1.4 特性开关 · E2E · 文档 · tag `v1.4.0`

**仍搁置**：SSH · 企业 SSO · VSIX 开放市场 · 全量 LSP/Pyright · 营销 rollout
