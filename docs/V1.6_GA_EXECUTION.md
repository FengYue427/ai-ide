## 4. 支付路线（Paddle 拒审后）

```mermaid
flowchart TB
  subgraph CN["国内用户"]
    A[支付宝 live ¥39/¥79] --> OK1[✅ v1.6 P0]
  end
  subgraph OS["海外用户"]
    B{Lemon Squeezy / Polar?}
    B -->|通过| C[MoR Checkout $9.99/$19.99]
    B -->|拒审或超时| D[GA: Free + BYOK · Pro 标注 Coming soon]
    C --> OK2[✅ v1.6 P0]
    D --> OK3[✅ v1.6 P0 备选]
  end
  P[Paddle 代码] --> S[保留 sandbox · 不切 live]
```

**推荐顺序**：
1. 先打通 **支付宝 live**（你已有 AppID + 代码）
2. 同步申请 **Lemon Squeezy**（个人通过率通常高于 Paddle）
3. 7 天内无 live → 海外 GA 走 **Free+BYOK**，v1.6.1 再接 MoR
