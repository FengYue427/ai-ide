## 2. Vercel 自定义域配置（如采用）

按 [CUSTOM_DOMAIN.md](./CUSTOM_DOMAIN.md)：

1. Vercel → Settings → Domains 绑定 `ide.example.com`
2. DNS 配置 CNAME/A，等待证书签发
3. Vercel 环境变量（Production）：
   - `APP_URL=https://ide.example.com`（无尾 `/`）
   - `DATABASE_URL` / `AUTH_SECRET` 保持不变

核对表： [V1.0.3_VERCEL_ENV.md](./V1.0.3_VERCEL_ENV.md)

## 3. OAuth / 支付回跳回归

### OAuth

- GitHub：`https://ide.example.com/api/auth/callback/github`
- Google：`https://ide.example.com/api/auth/callback/google`

要求：OAuth 控制台 redirect URI 与 `APP_URL` **一致**。

### 支付宝回跳

- 下单 → return_url 回到 `APP_URL` 域
- 回站后 plan 刷新（或提示刷新）正常
