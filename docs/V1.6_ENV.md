## 服务端（Vercel Production · v1.6 P0）

| 变量 | 说明 |
|------|------|
| `PLATFORM_DEEPSEEK_API_KEY` | 平台 AI Chat 网关（**新增 P0**） |
| `ALIPAY_*` | 国内 live · **无** `ALIPAY_SANDBOX` |
| `APP_URL` | `https://ai-ide-flame.vercel.app` |
| `BILLING_CRON_SECRET` 或 `CRON_SECRET` | 订阅到期 cron |
| `DATABASE_URL` | Neon |

**海外（v1.6.0 可选）**

| 变量 | 说明 |
|------|------|
| `PADDLE_*` | 保留 sandbox 联调；**不切 live**（Vendor 拒审） |
| Lemon Squeezy | **v1.6.1** 接入 |
