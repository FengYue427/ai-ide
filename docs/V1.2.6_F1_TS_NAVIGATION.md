# v1.2.6 F1 — Monaco TypeScript 语义导航

## 行为

- 编辑器模型 URI 与 `syncTypeScriptProject` 一致（`file:///`）。
- `registerCrossFileDefinition` / `registerCrossFileReference` 先调用 `getMonacoTypeScriptDefinitions` / `getMonacoTypeScriptReferences`，无结果时回退 `languageServiceHostCore` 启发式。
- 跨文件跳转仍通过 `inmemory://` 与 `registerEditorOpener` 打开标签页。

## 代码

- `src/editor/editorModelUri.ts`
- `src/editor/monacoTypeScriptNavigation.ts`
- `src/components/Editor.tsx` — `path={workspacePathToLibUriString(filename)}`

## 非目标

- 不启动独立 `tsserver` 子进程（仍用 Monaco Worker）。
