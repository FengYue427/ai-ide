## 2. 一键验收

```powershell
# 默认探测 https://ai-ide-flame.vercel.app
npm run ops:verify-p1

# 若本地 .env.local 已 mirror 生产变量（无 ALIPAY_SANDBOX）
npm run ops:verify-p1 -- --with-local-env

# 自定义域（绑定后）
$env:APP_URL="https://your-domain.com"
npm run ops:verify-p1

# 含 Cron 全链路（需 Vercel secret，勿提交仓库）
$env:APP_URL="https://ai-ide-flame.vercel.app"
$env:CRON_SECRET="<from Vercel>"
npm run ops:verify-p1
```

**包含**：`test:local` · 生产 smoke 5/5 · Cron 匿名 401 ·（可选）`verify-cron` · 法务页存在性 · env 清单（有 `.env.local` 时）
