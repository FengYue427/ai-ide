# 计划系统快速上手（v1.1.1）

> 5 分钟走完 **Plan → Spec → Chat → 报告**。设置入口：**齿轮 → 功能**。

---

## 1. 准备

1. 打开项目（或 `npm run dev` 本地）
2. 确认存在或可创建 `.aide/` 目录
3. Chat 保持 **Agent 模式**（默认开启）

---

## 2. 创建计划

**方式 A — 模板（推荐）**

1. 设置 → **Plan 管理**
2. 选择模板（功能开发 / Bug 修复 / 发布清单）
3. 输入标题 → **从模板创建**

**方式 B — Plan 模式**

1. Chat 开启 **Plan 模式**
2. 描述需求，AI 生成计划并写入 `.aide/plans/`

---

## 3. 执行计划步骤

1. 在 Plan 管理中勾选若干 `- [ ]` 步骤
2. 点击 **执行步骤** → 确认 → Chat 自动顺序执行
3. 完成后步骤在 plan 文件中变为 `- [x]`

快捷操作：

- **标记完成**：不跑 Chat，直接勾选完成
- **复制计划**：基于现有 plan 分叉新文件

---

## 4. 映射到 Spec（可选）

1. 先 **创建 Spec**（Specs 目录）
2. 在 Plan 中选择步骤 → **映射到 Spec** 或 **映射并执行**
3. Spec 任务会写入 `.aide/specs/<name>/tasks.md`，并记录溯源到 `.aide/meta/plan-spec-links.json`

溯源跳转：

- 计划步骤旁 **打开 Spec**
- Spec 卡片上 **来源计划** 按钮回到 plan

---

## 5. 任务队列（Chat 面板）

展开 **任务队列** 可：

- 查看当前执行、队列预览、成功/失败统计
- **复制报告** / **保存到 .aide/reports**
- **从最新报告恢复**（或设置 → 报告 → **预览恢复** → 确认入队）

队列刷新后会从 localStorage 恢复（schema v1）。

---

## 6. 报告与清理

设置 → **执行报告**：

- 打开、恢复队列、删除
- 多选 → 删除选中 / 导出 ZIP
- **清理旧报告**：保留最近 N 份

可开启「队列完成时自动保存报告」。

---

## 7. 工作区索引（重要）

若 AI 索引文件数与编辑器不一致：

1. 设置 → **计划总览**
2. 点击 **同步 .aide 到工作区索引**

仅同步 `.aide/**`，不覆盖其他文件。
