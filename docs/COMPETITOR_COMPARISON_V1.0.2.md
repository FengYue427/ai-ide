## 7. 与 Cursor 的「入门 70%」指什么

| 能力块 | Cursor 参照 | v1.0.2 达成度 | 缺口 |
|--------|-------------|:-------------:|------|
| 打开本地仓库 | 原生 | **85%** | 浏览器 500 限制 |
| Chat 问代码 | Chat | **75%** | 无全仓云上下文 |
| Agent 多文件改 | Composer | **60%** | 无 hunk、无强 Diff |
| Tab 补全 | Tab++ | **35%** | 单行为主 |
| 终端 | 集成 | **70%** | 浏览器 WebContainer |
| 订阅 | USD | **90%**（国内） | — |
| 插件 | 市场 | **5%** | 不做了 |

**结论**：**Agent + 本机盘 + 国内付费** 最接近；**Tab + 插件 + 调试** 不在 v1.0.x 范围。

## 11. 对外话术（可照读）

**可以说**：

- 浏览器打开就能 Agent 改本机项目，支持支付宝和自带 API Key。  
- 开源，适合学生、小项目、不能装 IDE 的环境。  
- Windows / **macOS** 可选桌面版：更大文件夹 + 本机终端。

**不要说**：

- 比 Cursor / Windsurf / Kiro 更强或替代。  
- 有 Copilot++、VS Code 插件、Specs 工程化。  
- 能无人值守改仓半小时。
