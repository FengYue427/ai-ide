## 1. 一句话定位

**1.0.4** = 在 **1.0.2 能力 + 1.0.3 运营稳定** 之上的 **「智能体验巩固版」**：

- 已交付能力 **产品化**（规则编辑入口、MCP 向导、Agent 上下文提示）  
- 竞品综合分 **2.75 → ~2.80**（仍非 Cursor 替代）  
- 为 **v1.1**（后台队列、协作、网关）预留架构空间，**本版不实现**
