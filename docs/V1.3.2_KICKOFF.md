## 1. 北极星

v1.3.0 加了**可观测性**（Tab 指标卡、插件计数），v1.3.2 让用户**用起来更顺**：

1. Tab 补全少打扰、少空请求  
2. 条件断点能设、能看、CDP 路径可测  
3. 第三方插件有**可复制的样例 + 文档**，运维卡不再只有空列表

仍 **不** 开 v1.4 的 Git 面板、2k 索引、FIM 生产默认全开。

## 2. 任务表（T1–T3）

### T1 — Tab 补全抛光

**现状**：`inlineCompletionService.ts` 已有 cache / FIM / platform / chat 路径；v1.3 F4 仅加指标卡。

| 交付 | 说明 | 文件 |
|------|------|------|
| 早退策略 | prefix &lt; N 字符或纯空白行不请求 | `inlineCompletionService.ts` |
| Debounce 默认值 | 生产推荐 300–450ms（可 `inlineCompletionPrefs` 配置） | `inlineCompletionPrefs.ts` |
| 空 suffix 处理 | 文件末尾光标：suffix 为空时仍走 cache key，不重复打 API | `buildCacheKey` |
| 指标路径标注 | FIM / platform / chat 失败原因细分（timeout / 413 / empty） | `inlineCompletionMetrics.ts` |
| 单测 | 早退 + cache hit + debounce 边界 | `inlineCompletionService.test.ts` |

**用户验收句**：在 `demo.ts` 里连续敲代码，Tab 幽灵字**跟手且不闪退**；设置页 Tab 指标「请求数 &gt; 0、失败率可读」。

**非目标**：FIM 默认强制开启 · 多行大块补全 · v1.4 P95 &lt;800ms 硬指标

---

### T2 — 条件断点完善

**现状**：`debugBreakpointPatterns.ts` 已支持 `condition` 注入 CDP；UI 有 glyph 样式与 i18n；inject 模式有单独 hint。

| 交付 | 说明 | 文件 |
|------|------|------|
| 断点列表编辑 | 侧栏断点项：编辑 condition、hitCount（已有字段则暴露 UI） | Debug 侧栏组件 |
| CDP 同步单测 | `buildCdpBreakpointAttempts` 含 condition 时 params 正确 | `debugBreakpointPatterns.test.ts` |
| hitCount 行为 | 未达 N 次前 continue（已有 `debugBreakpointHitCount.ts` 则补集成测） | 单测 |
| 文档一句 | [BROWSER_LIMITATIONS.md](./BROWSER_LIMITATIONS.md) 或调试卡：inject vs CDP 条件断点 | 文档 |

**用户验收句**：在可调试的 Node 样例里设 `x > 1`，断点 glyph 显示条件角标，命中条件才停。

**非目标**：全语言 DAP · 浏览器内 Wasm 调试 · 条件断点 E2E（可单测为主）

---

### T3 — 插件第三方案例

**现状**：官方插件 + 发布审核 API + 设置页 plugin-ops；缺「社区开发者从零走通」的固定样例。

| 交付 | 说明 | 位置 |
|------|------|------|
| Fixture manifest | 最小 community 插件（commands + 无特权） | `fixtures/plugins/community-sample/` |
| 走通文档 | 打包 zip → 提交审核 → pending 出现在 ops 卡 → 手动安装 | `docs/PLUGIN_COMMUNITY_SAMPLE.md` |
| API 夹具 | 本地 stack 可选 seed 一条 pending review（dev only） | `scripts/seed-plugin-review.mjs`（可选） |
| E2E 增强 | plugin-ops 在 seed 存在时断言 `pending ≥ 1`（无 seed 仍允许 0） | `e2e/plugin-ops.spec.ts` |

**用户验收句**：按文档 15 分钟内完成「样例插件提交 → 设置页看到 pending 计数 +1」。

**非目标**：开放公开上传 · VSIX · 自动过审 · 市场上架
