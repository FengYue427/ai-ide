## 1. 对账与 Cron（3.3-1）

### 每日抽验（约 10 分钟）

按 [BILLING_RECONCILE_DAILY.md](./BILLING_RECONCILE_DAILY.md)：

| # | 动作 |
|---|------|
| 1 | 支付宝商户后台昨日成功笔数 ↔ DB `PaymentOrder` paid |
| 2 | `npm run billing:reconcile -- <out_trade_no>` 抽查 1 笔 |
| 3 | Cron / `npm run billing:expire` 日志合理 |
| 4 | Vercel 日志无连续 notify 5xx |

### Cron 全链路

```powershell
$env:APP_URL="https://ai-ide-flame.vercel.app"
$env:CRON_SECRET="<Vercel CRON_SECRET>"
npm run billing:verify-cron
npm run ops:verify-p1
```

**记录**：在 [WEEKLY_OPS_TEMPLATE.md](./WEEKLY_OPS_TEMPLATE.md) 或运维笔记写日期 + 结果。
