# F4 — 侧栏/右栏 Resize + 窄屏折叠

> v1.2.2 第四阶段 · 对应 KICKOFF F4

## 目标

支持用户拖拽调整侧栏（FileSidebar）与右栏（RightPanel）宽度，并在窄屏下自动折叠侧栏以最大化编辑区。

## 实现方案

### 1. 持久化宽度偏好 — `src/lib/panelWidthPrefs.ts`

| 项目 | CSS 变量 | 最小值 | 最大值 | 默认值 |
|------|----------|--------|--------|--------|
| 侧栏宽度 | `--sidebar-width` | 180px | 420px | 260px |
| 右栏宽度 | `--right-panel-width-chat` / `--right-panel-width-git` | 280px | 600px | 360px |

- 使用 localStorage key `panel-width-prefs` 持久化
- 提供 `loadPanelWidthPrefs()` / `savePanelWidthPrefs()` / `clamp*()` 工具函数
- 通过 CSS 变量 `document.documentElement.style.setProperty` 实时应用宽度

### 2. 拖拽 Resize — `src/hooks/usePanelResize.ts`

基于底栏 resize 相同模式（Pointer Events）：

- `usePanelResize(width, onWidthChange, edge)` 返回 `onResizePointerDown/Move/Up/DoubleClick`
- `edge` 参数：`'left'`（右栏左边缘）或 `'right'`（侧栏右边缘）

### 3. Resize Handle 组件 — `src/components/PanelResizeHandle.tsx`

渲染 6px 宽的垂直分隔条：
- 悬停时显示 accent 色指示线
- 拖拽时高亮
- 双击恢复默认宽度
- 支持 `role="separator"` 和 `aria-label`

### 4. 窄屏折叠 — `AppShell.tsx`

```typescript
useEffect(() => {
  const mql = window.matchMedia('(max-width: 900px)')
  const handler = () => {
    if (mql.matches && (showChatPanel || showGitPanel || auxiliaryActive)) {
      setShowFileSidebar(false)
    }
  }
  handler()
  mql.addEventListener('change', handler)
  return () => mql.removeEventListener('change', handler)
}, [showChatPanel, showGitPanel, auxiliaryActive])
```

- 视口 ≤ 900px 且右栏打开时自动隐藏侧栏
- 右栏关闭时恢复侧栏显示
- 非侵入式 — 用户后续可通过 Activity Bar 按钮重新打开侧栏

### 5. CSS — `src/styles/workbench.css`

```css
.panel-resize-handle {
  flex-shrink: 0;
  cursor: col-resize;
  width: 6px;
  z-index: var(--z-local);
}
.panel-resize-handle:hover::after,
.panel-resize-handle:focus-visible::after {
  background: var(--accent-color);
  opacity: 1;
}
```

### 6. 状态管理 — `src/hooks/usePanelWidth.ts`

将持久化 + CSS 变量应用整合为一个 hook，供 AppShell 消费。
