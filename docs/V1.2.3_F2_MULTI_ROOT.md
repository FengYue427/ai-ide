# v1.2.3 F2 — 多根工作区 + 虚拟文件树

> **状态**：✅ 工程收口（2026-06-04）

---

## 交付

| 项 | 说明 |
|----|------|
| **Dev 默认开启** | `npm run dev` 下 `isMultiRootWorkspaceEnabled()` / 虚拟树为 true（`MODE !== 'test'`）；生产仍为 env 默认关 |
| **根隔离** | 切换根重置侧栏折叠/筛选；各根独立 `autosave-${id}` |
| **删除根** | `removeWorkspaceRoot` 异步清除对应 IndexedDB autosave |
| **activeFile** | `clampActiveFileIndex` — 删文件/缩列表时避免越界 |
| **Bootstrap** | 遗留 `autosave-default` 迁移；多 key 加载单测 |
| **E2E** | `e2e/multi-root.spec.ts` — 第二根独有文件 + 删除根 |

## 开关

| 环境 | 多根 | 虚拟树 (≥500 行) |
|------|:----:|:----------------:|
