# v1.5 环境变量与特性开关

> **世代**：v1.5.0 F0–F7 · 见 [V1.5_KICKOFF.md](./V1.5_KICKOFF.md) · GA：[V1.5_GA_EXECUTION.md](./V1.5_GA_EXECUTION.md)

| 变量 | 默认 | 说明 |
|------|------|------|
| `VITE_AI_GATEWAY` | dev 开 / prod **开** | 平台 AI 网关（F0） |
| `VITE_ALLOW_BYOK_LEGACY` | **关** | 云端 BYOK 旧入口（F0 弃用） |
| `VITE_TAB_PLUS_PLUS` | dev 开 / prod 关 | Tab++ 多行 ghost + FIM middle（F1/F2） |
| `VITE_AIDE_SPEC_ARTIFACTS_V2` | dev 开 / prod 关 | Spec 目录 · hooks.yaml 浏览/创建（F3） |
| `VITE_AIDE_RUNTIME` | dev 开 / prod 关 | Runtime 引擎：orchestrator + hookRunner（F4） |
| `VITE_AIDE_ACTIVITY_LINE` | dev 开 / prod 关 | Activity Line 生产 UI（F5） |
| `VITE_AIDE_RUNTIME_UI` | **关** | Activity Line / Runtime 设置卡 legacy stub |

## 生产推荐（Vercel · v1.5 GA）

```env
VITE_AI_GATEWAY=1
VITE_ALLOW_BYOK_LEGACY=0
VITE_TAB_PLUS_PLUS=1
VITE_AIDE_SPEC_ARTIFACTS_V2=1
VITE_AIDE_RUNTIME=1
VITE_AIDE_ACTIVITY_LINE=1
VITE_AIDE_RUNTIME_UI=0
```

仍沿用 v1.4 生产策略（Tab/FIM · 2k 索引 · Git hunk · MCP/插件）— 见 [V1.4_ENV.md](./V1.4_ENV.md)。
