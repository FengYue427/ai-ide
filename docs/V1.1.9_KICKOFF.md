## 3. API v2 摘要

```typescript
// manifest
{ "sdkVersion": 2, "permissions": ["ai", "debug:read", "ui"] }

// runtime
context.ai.getMode() // 'platform' | 'byok' | 'ollama' | 'unconfigured'
context.ai.complete(prompt) // 平台或 BYOK，与主应用一致
context.debug.getSummary() // 需 debug:read
```
