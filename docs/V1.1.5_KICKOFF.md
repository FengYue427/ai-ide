## 2. v1.1.5 北极星

## 3. 现状审计（代码锚点）

| 模块 | 路径 | 说明 |
|------|------|------|
| 终端 UI | `src/components/Terminal.tsx` | 200px 输出区，无 stdin |
| WebContainer | `src/hooks/useWebContainer.ts` | `runCommand` + `registerTerminalBridge` |
| 桌面桥 | `electron/main.mjs` · `desktopBridge.ts` | `spawn` 一次性，非 PTY |
| npm 解析 | `src/services/packageJsonService.ts` | `parsePackageScripts` |
| Palette 集成 | `src/components/CommandPalette.tsx` | `npmScriptCommands` |
| 任务服务 | `src/services/projectTasksService.ts` | parse `.aide/tasks.md` |
| 任务 UI | `src/components/ProjectTasksSection.tsx` | Settings 内预览 |
| 底栏开关 | `ideStore.showTerminal` · `EditorLayout` | 可扩展为多 tab 底栏 |
