## 1. 为何要有 1.0.5 + 1.0.5.x

| 原因 | 说明 |
|------|------|
| **1.0.4.x 已关闭** | 不再开 `1.0.4.5`；需新 semver 线 **`1.0.5`** |
| **v1.1 过大** | 队列/协作/网关不与运维债混 tag |
| **模式复用** | 同 1.0.4：**主版本叙事** + **第四段小步** |

**1.0.5** = 桥接主版本 Kickoff；**1.0.5.N** = 运营收口 → 质量 UX → v1.1 文档 → 世代收官。
