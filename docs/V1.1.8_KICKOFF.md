## 3. 生产环境变量

```bash
VITE_AI_GATEWAY=true
PLATFORM_DEEPSEEK_API_KEY=sk-...
# 可选
PLATFORM_AI_PROVIDER=deepseek
PLATFORM_OPENAI_API_KEY=sk-...
```

未配置平台 Key 时：网关返回 **503**，用户可切换 BYOK。
