## 长版（博客 / README 置顶）

### AI IDE v1.0 — 正式版上线

我们从 RC 公测进入 **正式版（D3）**：在保持「浏览器开箱、BYOK、国内支付」的同时，补齐可执行的订阅与付费说明。

**你现在可以：**

- 用 **工具 Agent** 多轮读写本机项目（list / read / write / search）
- 使用 **增量代码索引** 与 Tab 补全辅助编码
- 通过 **支付宝** 订阅专业版（¥19/月）或团队版（¥49/月），单次购买约 30 天权益
- 在订阅页管理 **取消续费 / 恢复 / 到期降级**（含 3 天宽限期）

**与 Cursor 的定位差异：** 我们优先服务「不能装桌面 IDE、需要人民币付费、自带模型 Key」的场景，而不是替代全语言 LSP 与 VS Code 插件生态。详见 [竞品说明](docs/COMPETITOR_SCORE_2026-05.md)。

**合规：** [服务条款](/legal/terms.html) · [付费与订阅说明](/legal/payment.html) · [隐私政策](/legal/privacy.html)

**已知限制：** 浏览器内运行（WebContainer）、本机文件夹建议 &lt;500 文件（桌面增强版在路线图中）。

感谢公测反馈，欢迎继续提 Issue。
