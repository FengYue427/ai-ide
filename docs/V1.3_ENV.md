# v1.3 环境变量与特性开关

| 变量 | 默认 | 说明 |
|------|------|------|
| `VITE_PYTHON_NAV` | dev 开 / prod 关 | Python F12 跨文件（符号索引） |
| `VITE_EMBEDDING_PERSIST` | dev 开 | 语义检索 embedding IndexedDB 缓存（7 天 TTL） |
| `VITE_INDEX_TELEMETRY` | dev 开 | 索引构建耗时写入设置页 |
| `VITE_BACKGROUND_AGENT` | **关** | 后台 Agent UI + API（见 V1.1_FEATURE_FLAGS） |

## 会话开关（localStorage）

| Key | 值 |
|-----|-----|
| `ai-ide:feature:pythonNav` | `1` |
| `ai-ide:feature:embeddingPersist` | `1` |
| `ai-ide:feature:indexTelemetry` | `1` |
| `ai-ide:feature:backgroundAgent` | `1`（仅 UI 演示，仍需服务端 env） |

## E2E

```javascript
localStorage.setItem('ai-ide:feature:pythonNav', '1')
```
