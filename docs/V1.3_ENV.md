# v1.3 环境变量与特性开关

| 变量 | 默认 | 说明 |
|------|------|------|
| `VITE_PYTHON_NAV` | dev 开 / prod 关 | Python F12 跨文件（符号索引） |
| `VITE_EMBEDDING_PERSIST` | dev 开 | 语义检索 embedding IndexedDB 缓存（7 天 TTL） |
| `VITE_INDEX_TELEMETRY` | dev 开 | 索引构建耗时写入设置页 |
| `VITE_BACKGROUND_AGENT` | **关** | 后台 Agent UI + API（见 V1.1_FEATURE_FLAGS） |

## 生产推荐（Vercel / 自托管）

> v1.3.1 起部署同学**仅按本表**配置，无需翻代码。详见 [V1.3.1_KICKOFF.md](./V1.3.1_KICKOFF.md) P4。

| 变量 | 生产建议 | 理由 |
|------|----------|------|
| `VITE_PYTHON_NAV` | **关**（`0` 或不设） | 启发式符号导航，非 LSP；小流量灰度可 `1` + 用户 localStorage |
| `VITE_EMBEDDING_PERSIST` | **开**（`1`） | 有 BYOK embedding 时降低重复请求与延迟 |
| `VITE_INDEX_TELEMETRY` | **开**（`1`） | 仅设置页只读耗时，无 PII |
| `VITE_BACKGROUND_AGENT` | **关**（`0`） | v1.3 仅观测卡，生产 API 未 GA |

### Vercel 示例（生产稳态）

```env
VITE_PYTHON_NAV=0
VITE_EMBEDDING_PERSIST=1
VITE_INDEX_TELEMETRY=1
VITE_BACKGROUND_AGENT=0
```

### 灰度 Python 导航（可选）

```env
VITE_PYTHON_NAV=1
```

用户还可在浏览器控制台启用会话开关（见下表），与 env 为 OR 关系。

## 会话开关（localStorage）

| Key | 值 |
|-----|-----|
| `ai-ide:feature:pythonNav` | `1` |
| `ai-ide:feature:embeddingPersist` | `1` |
| `ai-ide:feature:indexTelemetry` | `1` |
| `ai-ide:feature:backgroundAgent` | `1`（仅 UI 演示，仍需服务端 env） |

## E2E

```javascript
localStorage.setItem('ai-ide:feature:pythonNav', '1')
```

## 发版后 smoke

```bash
npm run smoke:production -- https://ai-ide-flame.vercel.app
```
