## 4. 验收（GA 黄金路径 · 二选一）

### 路线 A — 协作 M1

1. 用户 A 创建房间并分享链接/码  
2. 用户 B 加入，双方可见连接状态  
3. 双方各编辑不同文件 10 分钟，房间不丢、可重连  
4. Viewer 只读不可写（F3 后）  

### 路线 B — AI 网关

1. Pro 用户未配置 `apiKey`，选择「平台额度」  
2. Chat 单轮成功，用量 +1  
3. 超额后 429 + 引导 BYOK 或升级  
4. BYOK 用户行为与 v1.1.2 一致，无回归
