# v1.1 后台任务队列 — 数据模型占位（规划）

> **状态**：**已实现（v1.1.2 F1）** — 见 `prisma/schema.prisma` · `lib/api/handlers/jobs/`  
> **关联**：[ROADMAP_V1.1.md](./ROADMAP_V1.1.md) P1 · [V1.1.2_MASTER_PLAN.md](./V1.1.2_MASTER_PLAN.md)

---

## 目标

单用户、单仓库、可取消的后台 Agent 任务；结果回写工作区或通知前端。

---

## 表草案（Prisma 示意）

```prisma
model BackgroundJob {
  id          String   @id @default(cuid())
  userId      String
  status      String   // queued | running | succeeded | failed | cancelled
  repoKey     String?  // 工作区标识
  prompt      String   @db.Text
  progress    Json?
  result      Json?
  error       String?
  createdAt   DateTime @default(now())
  startedAt   DateTime?
  finishedAt  DateTime?
}
```

---

## API 草案

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/jobs` | 创建任务 |
| GET | `/api/jobs/:id` | 轮询状态 |
| POST | `/api/jobs/:id/cancel` | 取消 |
| GET/POST | `/api/jobs/process` | Cron Worker（Bearer `CRON_SECRET`） |

---

## 非目标（v1.1.0）

多仓库、企业审计、长时间无人值守（>30min）、PR 自动创建（可 v1.1.2）。
