## 3. 现状审计（代码锚点）

| 模块 | 路径 | 说明 |
|------|------|------|
| Git 服务 | `src/services/gitService.ts` | isomorphic-git：status / log / stage / commit / diff / push |
| Git 面板 | `src/components/GitPanel.tsx` | Changes + History tab · stage/unstage · 弹窗 diff |
| 编辑器同步 | `src/app/gitEditorSync.ts` | checkout / discard 后写回 `ideStore.files` |
| Diff 组件 | `src/components/DiffViewer.tsx` | Side-by-side / unified · Agent apply 复用 |
| 状态栏 | `src/components/StatusBar.tsx` | 分支名 + 变更数 → 打开 Git 面板 |
| Hook | `src/hooks/useGitStatus.ts` | 防抖 status + branch |
| 右栏 | `src/app/RightPanel.tsx` | GitPanel + `onShowDiff` → PanelHost DiffViewer |
| 测试 | `gitService.readWorktree.test.ts` · `gitEditorSync.test.ts` | worktree 同步 |

**已有能力（不需重写）**：init · stage/unstage all · commit · discard · branch checkout · HEAD vs workdir diff 数据。

**缺口（F1～F3 重点）**：

- 无 **编辑器 diff tab**（`EditorLayout` tabs 仅普通文件）
- staged diff（index vs HEAD）未单独展示
- history 项不可 drill-down
- 无 **SCM 输入框**与 VS Code 同级布局（commit 区在 staged 列表内，可抛光）
- 桌面本地文件夹 Git：Electron 项目路径 vs WebContainer FS 双轨（本包仍以 WebContainer FS 为主）
