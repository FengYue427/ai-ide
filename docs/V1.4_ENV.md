# v1.4 环境变量与特性开关

> **世代**：v1.4.0 F1–F7 · 见 [V1.4_KICKOFF.md](./V1.4_KICKOFF.md)

| 变量 | 默认 | 说明 |
|------|------|------|
| `VITE_TAB_FIM_PRODUCTION` | dev 开 / prod 关 | Tab 强制启用 + 300ms debounce（F1） |
| `VITE_INDEX_2K_PRODUCTION` | dev 开 / prod 关 | 2k 索引策略 + Worker 阈值 40（F2） |
| `VITE_GIT_HUNK_STAGE` | dev 开 / prod 关 | Git 块级 stage 面板（F3） |
| `VITE_DESKTOP_SHELL_PRODUCTION` | dev 开（桌面）/ prod 关 | 桌面壳自动恢复上次项目（F4） |
| `VITE_BACKGROUND_AGENT_PRODUCTION` | **关** | 后台 Agent 客户端可见（F5） |
| `VITE_MCP_PLUGIN_PRODUCTION` | dev 开 / prod 关 | MCP/插件官方目录生产策略（F6） |

## 生产推荐（Vercel）

```env
VITE_TAB_FIM_PRODUCTION=1
VITE_INDEX_2K_PRODUCTION=1
VITE_GIT_HUNK_STAGE=1
VITE_MCP_PLUGIN_PRODUCTION=1
VITE_BACKGROUND_AGENT_PRODUCTION=0
```
