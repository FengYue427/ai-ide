## 3. 终端（F1）

| 项 | 说明 | 优先级 |
|----|------|:------:|
| **xterm.js 嵌入** | 替换只读 `Terminal.tsx` 输出列表 | P0 |
| **WebContainer shell** | `spawn` 长进程、stdout/stderr 流式写入 xterm | P0 |
| **Electron 流式** | `desktop:run-command` 改为 chunk 推送或 node-pty | P0 |
| **terminalBridge** | 保持 `runTerminalCommand` API；增加 `getRecentTerminalLines` 供 Agent | P1 |
| **协作只读** | 协作 viewer 角色禁用 shell 输入 | P1 |
