## 1. 定位

**v1.1** = 在 **1.0.3 稳定运营** 之后的 **能力第二代际**，开始触碰 Cursor/Kiro 长期缺口中的 **可选** 项，仍 **不** 承诺 VSIX / 全语言 DAP / Tab++ 全量替代。

---

## 2. 候选方向（待 RFC 细化）

| 方向 | 说明 | 依赖 |
|------|------|------|
| **后台 Agent 队列** | 30min 级无人值守改仓 | 服务端队列、沙箱、计费 |
| **协作 M1** | 信令稳定 + 房间权限 | WebSocket/SFU |
| **AI 网关** | 平台 Key 可选（非 BYOK 独占） | 用量、风控、模型路由 |
| **i18n Phase 2** | 日文等 | 翻译与 E2E |
