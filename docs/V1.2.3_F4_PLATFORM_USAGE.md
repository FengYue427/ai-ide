# v1.2.3 F4 — 平台 AI 用量仪表盘

## 目标

登录用户在 **设置 → AI** 查看只读用量与估算成本，无需营销/账单系统。

## API

`GET /api/usage/dashboard`（需登录）

- `quota`：当日总配额（`ai_request` + `ai_platform_request`）
- `platformToday` / `otherToday`
- `daily[]`：近 7 日 UTC 分桶
- `costEstimate*Usd`：按 `PLATFORM_AI_ESTIMATE_USD_PER_REQUEST`（默认 `0.002`）估算

## 埋点类型

| type | 来源 |
|------|------|
| `ai_platform_request` | `POST /api/ai/chat` |
| `ai_request` | 客户端 `POST /api/usage/ai` 等 |

## 验证

```bash
npm run test:local
npm run build:api
```

登录后打开设置 → AI，应看到用量卡片与 7 日柱状图。
