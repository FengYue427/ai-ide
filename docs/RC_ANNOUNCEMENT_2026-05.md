## 中文

**AI IDE v1.0.0-rc.1 现已开放公测（路径 A · 不收款）**

在浏览器里编写、运行与对话的 AI 原生 IDE —— 无需安装桌面客户端。

**立即体验**：https://ai-ide-flame.vercel.app

### 你能做什么

- **BYOK（推荐）**：自带 OpenAI / Anthropic / DeepSeek 等 API Key，对话与补全直连服务商
- **可选云账号**：注册后同步工作区、查看今日 AI 用量
- **Monaco 编辑 + WebContainer 终端**、工作区上下文、Agent 应用 diff
- **中英界面**、9 种主题、Git 面板、插件沙箱（官方目录）
- **实时协作（Beta）**：实验性多人编辑

### 公测说明

- 当前为 **RC 公测**，**不收取任何费用**
- 应用内「订阅 / 专业版」仅用于配额与功能演示，不构成收费要约
- 云账号依赖部署方数据库配置；本站已启用 Neon + Vercel

### 文档与合规

- [隐私政策](https://ai-ide-flame.vercel.app/legal/privacy.html) · [服务条款](https://ai-ide-flame.vercel.app/legal/terms.html)
- 开源仓库：https://github.com/FengYue427/ai-ide
- 问题反馈：[GitHub Issues](https://github.com/FengYue427/ai-ide/issues)

### 反馈欢迎

注册登录、云工作区、英文界面、用量统计或 AI 对话任何问题，请带 **浏览器 + 操作步骤** 开 Issue。

## English

**AI IDE v1.0.0-rc.1 — public beta (Path A · no charges)**

A browser-first, AI-native IDE: edit, run, and chat without installing a desktop app.

**Try it**: https://ai-ide-flame.vercel.app

### Highlights

- **BYOK**: use your own API keys (OpenAI, Anthropic, DeepSeek, etc.)
- **Optional cloud account**: workspace sync and daily AI usage display
- Monaco editor, WebContainer terminal, workspace context, Agent apply flow
- Bilingual UI (zh-CN / en-US), themes, Git panel, plugin sandbox
- Realtime collaboration (**Beta**)

### Beta terms

- **No fees** during this RC
- In-app “Pro / subscription” labels are for quota demo only
