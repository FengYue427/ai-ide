## 1. 北极星

在 v1.2.9 工程 polish 之上，v1.3 用 **七个阶段** 把「日常主力 IDE」体验抬一档：

1. **多语言**：Python 跨文件 F12（符号索引，非全 LSP）  
2. **仓库理解**：embedding 持久化 + 索引构建遥测  
3. **Agent**：索引上下文注入 + 后台 Agent 可运维可见  
4. **补全**：Tab 指标可观测  
5. **插件**：审核列表运维增强  
6. **平台**：v1.3 特性开关与设置集中展示  
7. **GA**：文档、E2E、版本 1.3.0

**搁置**：SSH · 企业 SSO · VSIX · 营销 rollout
