# v1.1.3 环境变量

> **用途**：Kickoff 后在 Vercel Preview/Production 配置。  
> **规则**：仅启用**已拍板路线**相关变量；另一路线可留空。

---

## 共同（已有）

| 变量 | 说明 |
|------|------|
| `DATABASE_URL` | PostgreSQL |
| `AUTH_SECRET` / NextAuth 相关 | 登录 |
| `CRON_SECRET` | 后台任务 Cron（v1.1.2，与 v1.1.3 P0 独立） |

---

## 路线 A · 协作 M1

| 变量 | 必填 | 说明 |
|------|:----:|------|
| `VITE_COLLAB_M1_SIGNAL` | 客户端 | `true` 启用新信令路径（默认 `false`） |
| `LIVEKIT_API_KEY` | 若用 Livekit | 服务端签发 token |
| `LIVEKIT_API_SECRET` | 若用 Livekit | |
| `LIVEKIT_URL` | 若用 Livekit | `wss://...` |

自建 WebSocket 信令时改为：

| 变量 | 说明 |
|------|------|
| `COLLAB_SIGNALING_URL` | 服务端 WS 入口 |
| `VITE_COLLAB_SIGNALING_URL` | 客户端连接地址 |

---

## 路线 B · AI 网关

| 变量 | 必填 | 说明 |
|------|:----:|------|
| `VITE_AI_GATEWAY` | 客户端 | `true` 显示「平台额度」选项 |
| `PLATFORM_DEEPSEEK_API_KEY` | 首模型建议 | DeepSeek 平台转发 |
| `PLATFORM_OPENAI_API_KEY` | 可选 | OpenAI 兼容 |
| `PLATFORM_AI_DEFAULT_PROVIDER` | 可选 | 默认 `deepseek` |
| `PLATFORM_AI_RPM_PER_USER` | 可选 | 单用户每分钟上限（风控） |

**禁止**写入任何 `VITE_*` 平台 Key（仅服务端 `PLATFORM_*`）。

---

## 与 v1.1.2 后台 Agent 共存

| 变量 | 说明 |
|------|------|
| `BACKGROUND_JOB_WORKER_MODE` | `dummy` \| `agent` |
| `BACKGROUND_AGENT_API_KEY` | 云 Worker 专用，**不等同** Chat 网关 Key |

v1.1.3 网关 MVP 可不改后台 Worker；后续 patch 再统一平台 Key 策略。

---

## Preview 冒烟最小集

**路线 B**（更快验证）：

```env
VITE_AI_GATEWAY=true
PLATFORM_DEEPSEEK_API_KEY=sk-...
```

**路线 A**：

```env
VITE_COLLAB_M1_SIGNAL=true
LIVEKIT_API_KEY=...
LIVEKIT_API_SECRET=...
LIVEKIT_URL=wss://...
```
