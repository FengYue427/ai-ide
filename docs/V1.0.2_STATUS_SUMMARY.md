## 3. 当前能力清单（代码真实状态）

### 3.1 Web 产品

| 模块 | 能力 | 实现要点 |
|------|------|----------|
| 编辑器 | Monaco、多主题 | `AppShell` / `ideStore` |

| 账号 | 注册登录、云工作区 | Prisma + JWT Cookie |
| 订阅 | 支付宝、¥19/¥49、生命周期 | Path B、宽限期、对账脚本 |
| 插件 | 沙箱示例 + 目录 | `pluginSandbox` |
| 协作 | WebRTC Beta | `collaborationService` |
