## 建议补发（按性价比）

### 中文开发者社区

| 平台 | 形式 | 说明 |
|------|------|------|
| **SegmentFault 思否** | 专栏 | 与掘金类似，竞争小 |
| **开源中国 OSC** | 新闻/博客 | 开源项目上线常用渠道 |
| **HelloGitHub** | 投稿/自荐 | 适合「学生开源项目」叙事（需按当期规则投稿） |
| **少数派 SSPAI** | 投稿 | 偏效率工具，审核严，周期长 |
| **Linux.do** | 论坛帖 | 开发者氛围，类似 V2EX |

### 国际（可选，英文）

| 平台 | 说明 |
|------|------|
| **Hacker News** | Show HN: browser IDE + BYOK，勿强调国内支付 |
| **Reddit** | r/webdev r/selfhosted r/LocalLLaMA |
| **Product Hunt** | 需准备英文 + 截图 + 视频 |
| **Dev.to** | 短文 + 链接 |

### 社交 / 短视频

| 平台 | 说明 |
|------|------|
| **抖音** | 复用 B 站 60s 剪辑，竖屏 |
| **视频号** | 朋友圈扩散 |
| **Twitter/X** | 英文一句 + 链接 |

### 不优先

| 平台 | 原因 |
|------|------|
| 微信公众号 | 需主体与审核，周期长 |
| 贴吧 | 易沉，维护成本高 |
| Telegram 群 | 国内触达有限 |

## 统一话术（全平台）

- **是什么**：浏览器 AI IDE，Agent 改本机文件夹  
- **差异**：开箱、BYOK、开源；**不声称碾压 Cursor**  
- **链接**：站点 + GitHub +（可选）桌面 Release  
- **学生向**（小红书）：课余项目、十六岁、练手 — **不提价格**
