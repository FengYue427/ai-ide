## 正文（直接粘贴）

从去年开始课余折腾，今年刚满十六岁，把一个小项目做成了正式版 ✨

它叫 **AI IDE**——在浏览器里打开，就能让 AI 帮你改**电脑上的项目文件夹**，不用先装很重的 IDE。

为什么做这个？
· 学校/机房电脑不方便装一堆软件
· 想练 Agent 改真实文件，而不是只能玩在线片段
· 模型 Key 想放在自己这边（本地保存，自己填 DeepSeek 等）

现在能玩到什么：
✅ 打开本机文件夹，AI 能读能写能搜
✅ Agent 多轮改代码，带改动预览
✅ 浏览器里就能跑起来；Windows 还有便携版，大一点的仓库 + 本机命令行

🔗 在线试试
https://ai-ide-flame.vercel.app

💻 开源地址（欢迎 Star / Issue）
https://github.com/FengYue427/ai-ide

Windows 便携版（可选）
https://github.com/FengYue427/ai-ide/releases/latest

不是要和 Cursor 硬刚啦～更像是「浏览器开箱 + 自己带 Key」的入门练习向工具。
我还在上学，更新可能不会特别快，但会认真看留言和 Issue。

如果你也在学编程，欢迎玩玩看，有问题评论区问我～
