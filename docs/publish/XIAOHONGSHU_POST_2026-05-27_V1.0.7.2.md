## 正文（可直接发）

这段时间我一直在打磨一个开源 AI IDE。  
如果一句话总结 **V1.0.2 → V1.0.7.2**：  
**从“功能有了”，变成“日常真的能顺着用”。**

我主要做了 5 件用户能直接感受到的事：

1) **聊天更可控**  
- 生成时可以随时停止，不用干等  
- Agent 模式默认打开，开箱就能用  
- 请求太大（413）会提前提醒，还能一键“精简后再发”

2) **文件区终于像个文件管理器**  
- 左侧从“平铺”改成“文件夹树”  
- 支持展开/折叠，文件多的时候不会一团乱  
- 文件数和上限会直接显示，心里有数

3) **大项目体验更稳**  
- 以前会遇到“自动保存把工作区覆盖掉”的坑，这块已经修了  
- 浏览器和桌面版的文件上限规则也统一了（不再神秘卡 100）

4) **Agent 不再黑盒**  
- 现在能看到它调用了什么工具、成功还是失败  
- 也补了常用文件操作（移动、删除、建目录），不只是“说一堆”

5) **插件和整体 UI 都更像产品了**  
- 插件目录从少量 demo 到可筛选、可评分  
- Chat 区做了 v2 样式，信息层次更清楚，代码块更好读

---

我自己最在意的一点是：  
**不是堆功能，而是把“卡手感”的地方一个个抹平。**

后面会继续做聊天完成度（比如更好的消息操作和展示），  
但这版先把“顺手”和“稳定”放在第一位。

如果你也在做 AI 产品，欢迎交流你最头疼的体验问题 👇

## 结尾短版（可替换）

从 1.0.2 到 1.0.7.2，最大的变化不是“更复杂”，而是“更顺手”。  
我会继续按这个方向慢慢打磨。
