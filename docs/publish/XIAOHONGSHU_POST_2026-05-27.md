## 正文

今天主要在做 **v1.0.6** 这一串（1.0.6.1 → 1.0.6.4），方向很简单：**别再做用户看不见的东西**，改界面里能摸到的能力。

---

**Agent（1.0.6.1）**

以前 Agent 调工具，Chat 里只有一行行字，不知道成功没、改了几个 hunk。

现在 Chat 里加了 **工具调用面板**，可以折叠展开：工具名、摘要、成功/失败、hunk 数量，点一下还能看原始输出。

另外给 Agent 补了三个文件工具：
· `move_file` 移动/重命名  
· `delete_file` 删除  
· `create_dir` 建目录  

和本机工作区、磁盘会同步，不是只改内存。

---

**文件管理器（1.0.6.2）**

工作区左侧文件树终于能当「真文件管理器」用了：

· 右键：重命名、移动、删除、新建子文件夹  
· **F2** 重命名、**Delete** 删除  
· 工具栏可以新建文件夹  

改完文件会自动 **重建项目索引**，不然 @ 检索会对不上。

---

**插件（1.0.6.3）**

官方目录从 2 个 demo 扩到 **6 个**：

· JSON 格式化  
· TODO / FIXME / HACK 扫描  
· 工作区行数统计  
· Markdown 弹窗预览  

市场 Tab 加了 **标签筛选**（格式化、效率、Markdown 等），每款有个 **★ 分** 徽章，找插件不用瞎翻。

---

**1.0.6.4**

主要是文档和版本收口：Release 说明、竞品表更新到 **~2.90**（相对 1.0.4 的 ~2.80 涨了一档，还是入门档，不是 Cursor 那挂）。

单测 **222** 个，本地全绿。
