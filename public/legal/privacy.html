隐私政策

版本：v1.0.0-rc.1 · 最后更新：2026 年 5 月 24 日

公测说明（路径 A）：当前为 RC 公测，不收取费用。云端账号与工作区同步为可选功能；您也可仅使用 BYOK（自带 API Key）与本地编辑，无需注册。

适用范围

本政策适用于您通过我们部署的 AI IDE 网站（例如 https://ai-ide-flame.vercel.app）及同源 API 使用可选云端服务的情形。自托管实例由相应运营方负责。

我们收集的信息

  • 账号信息：注册时的电子邮箱、显示名称（可选头像 URL）。
  • 云工作区：您主动保存到云端的项目文件摘要、编辑器设置等（以您提交为准）。
  • 用量记录：用于公测配额展示的 AI 请求计数；不包含您的 API Key、模型对话正文或代码内容（对话直连您选择的 AI 服务商）。
  • 会话 Cookie：用于保持登录状态的 auth-token（HttpOnly）。
  • 技术日志：托管平台可能记录 IP、请求时间、错误信息（用于安全与排障）。

默认保存在您设备本地的信息

  • 第三方 AI 服务商的 API Key、编辑器主题与语言偏好等（浏览器 localStorage / IndexedDB）。
  • 未登录时的本地工作区与离线用量计数（若您未启用云同步）。

我们不通过服务器处理的内容

您配置的 API Key 与 AI 对话内容直接发往所选 AI 服务商，不经由我们的应用服务器代理或存储（除非未来单独公告变更）。

存储与托管

  • 云端数据库：PostgreSQL（例如 Neon），区域以服务商为准。
  • 应用托管：例如 Vercel（静态资源与 Serverless API）。
  • 实时协作（Beta）：若启用，可能通过 WebRTC 在对等端之间传输编辑数据；请参阅服务条款。

处理目的与法律依据（摘要）

我们处理上述数据用于：提供账号与工作区同步、配额统计、安全审计、改进稳定性。公测期间不进行付费结算。

保留期限

账号存续期间保留相关数据。您可申请删除账号及云工作区；删除后我们将在合理期限内从生产库中清除，备份副本可能短期保留。

您的权利

您可请求查阅、更正、导出或删除账号及关联云数据。请通过下方联系方式提交，我们将在合理期限内答复。

未成年人

本服务面向具备完全民事行为能力的用户。若您未满当地法定年龄，请勿注册云端账号。

政策变更

我们可能更新本政策；重大变更将在站内或仓库公告。继续使用云端服务即视为接受更新后的政策。
