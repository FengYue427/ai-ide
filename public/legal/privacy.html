隐私政策

最后更新：2026 年 5 月 · 本页为开源产品模板，正式运营前请由法务审阅。

我们收集什么

  • 账号信息：注册时的邮箱、显示名称（若使用云端登录）。
  • 工作区数据：您选择保存到云端的项目文件与设置。
  • 用量记录：用于配额管理的 AI 请求计数（不含您的 API Key 与模型对话正文）。
  • 本地数据：AI 提供商 API Key、编辑器偏好等默认保存在浏览器本地，除非您启用云同步。

我们不收集什么

您配置的大模型 API Key 与对话内容直接发往所选 AI 服务商，不经过我们的服务器代理（除非未来单独公告）。

数据存储

云端数据存储在受托管的 PostgreSQL 数据库（如 Neon / Supabase）。本地数据存储在浏览器 IndexedDB / localStorage。

您的权利

您可请求导出或删除账号及关联工作区数据，请联系项目维护者（见仓库 README）。

联系

问题与请求请通过 GitHub Issues 提交。
